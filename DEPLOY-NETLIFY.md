# Netlify Deployment

This is the simplest way to get the trivia site online.

## 1. Prepare the folder

Before uploading, make sure [config.js](./config.js) contains your Supabase values.

## 2. Create a Netlify site

1. Sign in to Netlify.
2. Go to the Sites page.
3. Drag the full `Juneteenth 2026` folder into the deployment area.

## 3. Share the URL

Netlify will assign a live URL immediately. You can:

- keep the generated `netlify.app` URL
- or rename the site in Netlify settings for a cleaner link

## 4. Update later

If you make changes, drag the folder in again to redeploy.
